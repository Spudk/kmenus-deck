# KMENUS Deck

KMENUS 투자/사업 제안서 — 온라인 슬라이드 덱.

## 링크
GitHub Pages 활성화 후: `https://<아이디>.github.io/kmenus-deck/`

## 파일
- `index.html` — 메인(투자자용) 덱
- `deck-assets/` — 표지·앱·창업자 사진 (공유 자산)
- `deck-stage.js`, `image-slot.js` — 덱 엔진

## 오디언스별 변형 만들기
`index.html`을 복제해 이름만 바꾸면 새 변형이 됩니다:
- `deck-gov.html` — 국가지원사업용
- `deck-accelerator.html` — 액셀러레이터용

각 파일은 같은 `deck-assets/`·스크립트를 공유하므로, 사진은 한 번만 올리면 됩니다.

## 수정 방법
GitHub에서 파일 열기 → ✏️(Edit) → 내용 수정 → **Commit changes** → 1~2분 뒤 사이트 자동 반영.

## 발표
링크를 열고 좌우 화살표(← →)로 슬라이드 이동. `P`로 PDF 인쇄.
